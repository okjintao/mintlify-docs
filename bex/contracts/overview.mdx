## Core Contracts

BEX consists of several core smart contracts that work together to provide decentralized exchange functionality.

### BexRouter

The main entry point for all trading operations.

**Address (Testnet)**: `0x...` (Coming Soon)
**Address (Mainnet)**: `0x...` (Coming Soon)

**Key Functions:**
- `swapExactTokensForTokens()` - Swap exact input amount
- `swapTokensForExactTokens()` - Swap for exact output amount
- `addLiquidity()` - Add liquidity to pools
- `removeLiquidity()` - Remove liquidity from pools

### BexFactory

Creates and manages liquidity pools.

**Address (Testnet)**: `0x...` (Coming Soon)

**Key Functions:**
- `createPair()` - Create new trading pair
- `getPair()` - Get pair address for two tokens
- `allPairs()` - List all pairs

### BexPair

Individual liquidity pool contract for each trading pair.

**Key Functions:**
- `swap()` - Execute swap within pool
- `mint()` - Mint LP tokens (add liquidity)
- `burn()` - Burn LP tokens (remove liquidity)
- `getReserves()` - Get current token reserves
